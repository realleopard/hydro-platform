# Hydro Platform Frontend

流域水系统模拟模型平台前端应用

## 技术栈

- React 18
- React Router 6
- Cesium (3D 可视化)
- React Flow (工作流编排)
- Recharts (图表)
- Zustand (状态管理)

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 功能模块

- **模型库**: 浏览、搜索、管理水文模型
- **工作流**: 可视化编排模型计算流程
- **可视化**: 3D 数字孪生、数据渲染
- **教学**: 虚拟实验室、参数分析
